# Simulation Overview

This simulation models a 2D drone navigating an environment using FMCW radar. 

- **Drone**: moves forward with lateral collision avoidance.
- **Obstacles**: random movement with bounds.
- **Radar**: simulates beat signal, computes Range-Doppler map.
- **Visualization**: dual plot - environment map and radar view.
- **Modular functions**: for initialization, update, detection, and plotting.
