# 2D Autonomous Drone Radar Simulator

This repository contains a MATLAB-based simulation of a 2D autonomous drone using FMCW radar for obstacle detection and avoidance.

## Features
- Range-Doppler radar simulation
- Multiple moving obstacles
- Real-time collision avoidance
- Dual visualization: 2D environment + Range-Doppler map
- Modular, professional code structure

## Getting Started
1. Open MATLAB
2. Navigate to `src/`
3. Run `main_simulation.m`
